## ‚ùì FAQ

### General Questions

**Q: Does NoteDrop work without Obsidian?**
A: Yes! NoteDrop works with any folder structure. Just choose "Simple Folder" mode and point it to where you want your notes saved as markdown files.

**Q: Is there an iOS version?**
A: Not yet. NoteDrop is currently Android-only. iOS version is being considered based on demand.

**Q: Does it require internet?**
A: No. NoteDrop works completely offline. All sync happens locally on your device.

**Q: How much does it cost?**
A: Core features are free forever. Optional premium features (like voice transcription) will be available in the future.

### Obsidian Integration

**Q: Will this modify my vault structure?**
A: NoteDrop only creates notes in the locations you specify. It never reorganizes, deletes, or modifies existing vault structure.

**Q: Does it work with Obsidian Sync?**
A: Yes! NoteDrop writes to your local vault. Obsidian Sync handles cloud synchronization as normal.

**Q: Can I use custom daily note formats?**
A: Yes! NoteDrop reads your `.obsidian/daily-notes.json` config and respects your date format, folder location, and templates.

**Q: What if my daily note doesn't exist yet?**
A: NoteDrop creates it automatically using your vault's daily note format and template (if configured).

### Privacy & Security

**Q: Where is my data stored?**
A: Only on your device. Notes are captured to local storage and synced to your vault folder. Nothing is sent to servers.

**Q: Do you collect any data?**
A: No. NoteDrop includes zero analytics, tracking, or data collection. Your notes are completely private.

**Q: What permissions does it need?**
A: Storage access permission (to read/write your vault). Optional: microphone (for voice capture), camera (for photo capture).

**Q: Is it open source?**
A: [If open source: Yes! Source code is available on GitHub.] [If not: Not currently, but we're considering it.]

### Technical Questions

**Q: How fast is "instant" capture?**
A: From widget tap to saved note: under 2 seconds typically. Depends on device speed and vault location.

**Q: What file formats are supported?**
A: Markdown (.md) files. Text notes are saved as markdown with optional formatting.

**Q: Can I sync multiple vaults?**
A: Currently one vault at a time. Multiple vault support is planned for future premium features.

**Q: Does it work with Obsidian plugins?**
A: NoteDrop writes standard markdown files. Any Obsidian plugins that work with your notes will work normally.

### Troubleshooting

**Q: Widget says "Loading" forever?**
A: Make sure you've configured vault path in settings and granted storage permissions. Try removing and re-adding widget.

**Q: Notes aren't appearing in daily notes?**
A: Check that your daily note header matches settings exactly (including ## symbols). Verify vault path is correct.

**Q: "Permission denied" error?**
A: Re-grant storage permissions in Settings ‚Üí Apps ‚Üí NoteDrop ‚Üí Permissions. You may need to reselect vault folder.

**Q: Voice recording not working?**
A: Grant microphone permission in Android settings. Check device storage isn't full.
